Core Changes

What changes

  • Purple → Zinc/Slate grayscale base
  • White bg → subtle #fafafa background
  • Rounded-lg buttons → pill-shaped, high-contrast
  • Card shadows → hairline borders (#e5e5e5)
  • Typography: Inter, tighter leading, cleaner hierarchy

What stays

  • Same page structure & routes
  • Same component tree (shadcn/ui)
  • Same data model & APIs
  • Dark mode still supported
  • Tailwind CSS (just new tokens)
