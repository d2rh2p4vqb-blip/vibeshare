AI 音乐生成器

输入歌词和风格偏好，AI 自动生成完整歌曲，支持多种流派

Claude Suno Vercel
2 小时前
▲ 128 点赞
★ 收藏
访问作品 →

这是一个基于 Claude Sonnet 和 Suno API 的 AI 音乐生成工具。用户可以输入歌词、选择音乐风格（流行、摇滚、电子、古典等），系统会自动生成带有完整编曲的歌曲。

项目使用 Next.js 构建前端，通过 Vercel 部署，后端调用多个 AI 模型 API 实现从文字到音乐的完整转换流程。

创作 PROMPT
Build a music generation app. Users input lyrics + genre. The app calls Suno API with a structured prompt combining: mood analysis from Claude, genre classification, and lyrical structure. Output: full song with cover art.
